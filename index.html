Why you'll love it

🔒 Privacy first

Your emails, calendar, and files never leave your Mac. No data is sent to any server.

🤖 Smart automation

Let AI handle repetitive tasks: sort mail, schedule meetings, run scripts with natural language.

💬 Chat integrations

Use it from WhatsApp, Telegram, Discord or Slack — it works like a personal assistant in your favourite messenger.

⚡ Lightweight

Written in efficient code, minimal resource usage. Perfect for background operation on Mac.

🚀 One‑line install (macOS)

# Clone and run installer
git clone https://github.com/JudeCampbell1/PersonalTaskAgent.git
cd PersonalTaskAgent
chmod +x install.sh
./install.sh

Works on Intel and Apple Silicon (M1/M2/M3). After installation, launch from terminal with pta or integrate with your chat apps.
